This time we’re aiming to cheat and complete a game on the extreme difficulty. We also have to get our wins to 9999.
This game looks like a variant of Simon Says where the goal is to remember a sequence of colours.
You could just play the game and win 9999 times and try your luck on the extreme difficulty, but what would fun would that be? ;)
Walkthrough
First, let’s try to tackle the challenge of getting our wins to 9999 or more. We have to find out how the number of wins is stored for each player.
Start by taking a look at the page source to identify how the game runs.
<script src="../ex10_assets/js/app.js"></script>
If you’ve gone through the source carefully, you should notice the line above. The game looks to be running on JavaScript with the source code of the game in the app.js file above.
Now we have to examine how the game stores our number of wins. One particular function caught my attention.
finishGame: function(action) {
    if (action === "win") {
      ...
      ++this.data.stats.wins;
      ...
    }
    ...
    localStorage[this.data.username] = JSON.stringify(this.data.stats);
    ...
},
I’ve removed most of the code for simplicity, but if you go through the whole function, you should also see the code that handles completing the game on the extreme difficulty.
I’ll just briefly explain the code above:
  1. if the player has won the game
    1. increase the number of wins by 1
  2. store all stats(including number of wins) in the localStorage for the current player(based on username)
From the code, we know that our number of wins are stored in localStorage. I’ll be using Developer Tools in Google Chrome to modify the app.js file to cheat the game into giving me 9999 wins.
finishGame: function(action) {
    if (action === "win") {
      ...
      this.data.stats.wins = 9999;
      ...
    }
    ...
    localStorage[this.data.username] = JSON.stringify(this.data.stats);
    ...
},
I’ve modified the line of code that increases the number of wins by 1, to set the number of wins to 9999. I just need to win the game once and I’ll get 9999 wins. Of course, there are many other ways to cheat the game into giving you 9999 wins, this is just the method I’ve chosen.
Next, let’s take a look at how the game determines if we win at the extreme difficulty.
finishGame: function(action) {
    if (action === "win") {
        if (this.data.totalShownColors === 9) {
            setTarget({
              extreme: true
            });
        }
        ...
    } 
    ...
},
Once again, removed some code for simplicity. A brief explaination of the code:
  1. if the player has won the game
    1. if the number of colours shown is equal to 9
      1. ‘player has completed the game on the extreme difficulty’ is set to true
Again, there are many ways to change the code to cheat and win the game. I’ve chosen to change the first if action condition to “lose”. Combine this with my modification to get 9999 wins and all I need to do is lose a game on extreme difficulty to complete this level.
Using Developer Tools, you can modify the code to your own liking. When done, just press Ctrl+S (on Windows) to save the script.
My final finishGame function:
finishGame: function(action) {
if (action === "lose") {
    if (this.data.totalShownColors === 9) {
        setTarget({
          extreme: true
        });
    }
    this.data.stats.wins = 9999;
    this.addAlert("Congrats. You won");
} else if (action === "lose") {
    ++this.data.stats.loses;
    this.addAlert("Oops. You did not make it...");
}
this.data.stats.pts += this.data.points;
localStorage[this.data.username] = JSON.stringify(this.data.stats);
return this.updateStats();
},
Lose a game on the extreme difficulty and level completed!
Success!
